What is an SSL and why is it so important?

SSL stands for secure socket layer. An SSL encrypts information sent to a web server from a website visitor, which prevents hackers from intercepting and using that information to steal that person’s identity. A website that contains an active SSL will feature a padlock and “https” preceding the domain name in the address bar of the browser, so anyone who visits your site will know if you have an active SSL. If you do not have an SSL or it has expired, we take action to ensure the website displays as secure.
